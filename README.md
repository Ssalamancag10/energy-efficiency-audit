# Energy Efficiency Audit – Compressed Air Leak Analysis

This project simulates an industrial energy audit focused on compressed air leaks.

The objective is to estimate airflow losses and financial impact based on ISO 11011 standards.

Technologies used:

- Python
- Pandas
- NumPy
- SQL
- Power BI

Project workflow:

Raw Data → Python ETL → SQL Database → Data Analysis → Dashboard → Executive Report

Key results:

• 518 leak points analyzed
• Estimated annual loss: $320,000
• Top 10 leaks represent 40% of losses

Skills demonstrated:

- Data cleaning
- Engineering calculations
- ETL pipeline development
- SQL data analysis
- Business intelligence dashboards
